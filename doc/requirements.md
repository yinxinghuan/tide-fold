# Tide Fold 需求文档

## 1. Overview

Tide Fold 是一款 6–12 秒完成一轮的 WebGPU 感官玩具：玩家单指旋转观看 70,000 个 MLS‑MPM 水粒子，用双指向内捏合把无形容器折向中央，松手后观看同一片海潮反冲并自动归静。

玩家主要来自 Aigram 手机信息流，首次停留通常低于 15 秒；核心承诺是无需理解规则，一次横向拖动就能亲手挤压一整块有重量的海水。

## 2. Visual Design

- 原作的深蓝屏幕空间流体、cubemap 折射、固定斜俯相机和实体水体是唯一视觉主角。
- 产品模式移除原调试表单、粒子开关、GitHub 图标与说明文字；左上仅保留标题和实验编号，底部仅显示一个当前动词。
- 幽灵引导使用 Google Material `touch_app` 轮廓，必须同步改变真实容器宽度和真实水体。
- 目标画布为 390×844 与 320×568 竖屏；桌面可铺满，但不增加桌面专属控制面板。
- `?baseline=1` 恢复原 UI、MLS‑MPM 70,000 粒子、相机拖动和箱体滑杆，以供逐帧对照。

## 3. Game Mechanics

- 默认使用原作 MLS‑MPM 模式与 70,000 粒子，初始箱体为 `[40, 30, 60]`，初始距离 70，模拟每帧执行 2 个 compute step。
- 产品状态为 `loading → idle → folding → release → recovery → idle`。
- 双指开始时记录间距，向内捏合累计收缩；折叠强度 `fold = clamp((startDistance-currentDistance) / (viewportWidth × 0.32), 0, 1)`。
- 原箱体 Z 宽度比例从 `1.0` 映射到 `0.52`；压缩仍遵守原作每帧最小闭合速度 `-0.007`，扩张速度在松手后限制为每帧 `+0.018`，避免瞬时穿透。
- 折叠超过 `0.72` 进入 ready；松手后保持峰值 180 ms，再用 1,650 ms 恢复到比例 1.0，水体随后继续按原物理自行归静。
- 不自动演示；底部只说明“双指捏合海潮”这一隐藏手势，首次有效捏合后淡出。
- 不增加泡沫、碎片、额外粒子、屏幕震动或第二套波浪；峰值完全来自原物理压缩和流体渲染。

## 4. Controls

- Touch / Pointer：单指或鼠标拖动旋转原作相机，无需文字说明。
- Hidden gesture：双指向内捏合控制折叠强度，任一触点松开或取消时触发释放与恢复；`touch-action: none`阻止页面滚动抢走手势。
- Baseline 模式保留原鼠标拖动相机、滚轮缩放、箱体滑杆和模拟选项。
- Keyboard：无必需键盘操作；WebGPU 错误页提供一个 44×44 以上的“重新尝试”按钮并支持 Enter/Space。

## 5. Win / Lose Conditions

本作没有胜负、分数或排行榜。一次闭环在容器恢复到宽度比例 `0.985` 且 release 后至少经过 1,650 ms 时完成，水体和两侧边界线自身承担状态反馈，不用文字逐步播报。WebGPU 不可用、adapter 获取失败或 device lost 时停止循环，显示错误原因与重新尝试入口，不伪造降级海水。

## 6. Sound Effects

- 首次真实按下：100 Hz 正弦波，70 ms，音量峰值 0.035，快速淡出。
- 折叠过程：由拖动速度驱动的 80–180 Hz 正弦声部，最高音量 0.044，同一时间只保留一个持续声部。
- 超过 0.72：260 Hz 三角波，90 ms，音量 0.025，提示临界点。
- 松手反冲：90→46 Hz 正弦下滑，420 ms，音量峰值 0.07。
- 仅在真实用户输入后创建或恢复 AudioContext；幽灵演示静音，音频失败不阻塞视觉。

## Interface brief

- 主要用户：信息流中的手机用户；单次 6–15 秒。
- 唯一动作：横向拖动并松手；记忆画面是深蓝水体被折成一道厚重竖脊后反冲。
- 状态：loading、guide、idle、folding/ready、release/recovery、WebGPU error/device lost；无菜单与结果页。
- 安全区：顶部 `max(20px, env(safe-area-inset-top))`，底部 `max(24px, env(safe-area-inset-bottom))`；HUD 不经过画布中央 60% 的手势带。
- 双语：根据 `game_locale` 或浏览器语言选择 zh/en；提示限制为 4 个英文词或 8 个中文字。
